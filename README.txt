PHP Shoutit! Project
Ed Friesema 11/24/2016

Objective :  To create a simple text chat forum that stores chat messages, time submitted, and user ID's in a Maria DB database, and displays them in any browser.

Platform :Developed on Windows 10 using Aptana Studio 3 IDE

Using XAMPP server
Apache 2.4.17 Win32
PHP version 5.6.23
Database :  /maria DB 10.1.13

Files 
index.php-  main input/output page read “shouts” database and displays 5 most recent messages
database.php- establishes connection with Maria SDB database”Shouts”;
process.php – Governs what happens when you hit the Submit button.  It records the chat,username and time submitted into a “shout” database
style.css-  creates look for display and allows the application to be responsive to screen size changes
